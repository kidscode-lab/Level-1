📦 Python Data Types

Different types of data need different "containers" in Python

🔢

Integer (int)

Whole numbers (positive, negative, or zero)

42
-17
0
1000
Perfect for: Age, count of items, scores
🎯

Float

Decimal numbers (numbers with a dot)

3.14
-2.5
0.0
99.99
Perfect for: Prices, measurements, percentages
📝

String (str)

Text wrapped in quotes (single or double)

"Hello World"
'Python is fun!'
"123"  # This is text, not a number!
"🐍"
Perfect for: Names, messages, any text
✅

Boolean (bool)

Only two values: True or False

True
False
Perfect for: Yes/no questions, on/off switches
