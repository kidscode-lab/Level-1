📦 What are Variables?

Variables are like drawers in memory that store your data!

🗃️

Think of Variables Like:

  • 📦
    Storage Boxes
    Each variable is like a labeled box that holds one piece of data
  • 🏷️
    Name Tags
    Every variable has a name so you can find your data later
  • 🔄
    Changeable Content
    You can change what's stored in a variable anytime
💡

Simple Example: Think of myage = 15 as putting the number 15 into a box labeled "myage"

✨ F-Strings: The Magic Way to Use Variables

F-strings make it super easy to put variables inside sentences!

👩‍💻

Your Code

Create a variable

myage = 15
→
🎯

F-String

Put it in a sentence

print(f"Hello! I'm {myage} years old!")
→
🖥️

Output

Beautiful result!

Hello! I'm 15 years old!
🌟

Pro Tip: Always use f-strings instead of old-style string concatenation - they're easier and faster!

🔄 Converting Input to Numbers

input() always gives you text - but what if you need numbers for math?

⚠️

The Problem

input() always returns a string (text), even if you type numbers!

# This won't work for math!
age = input("How old are you: ")
# age is now "15" (text), not 15 (number)
✨

The Solution

For whole numbers: int()
age = int(input("How old are you: "))
For decimal numbers: float()
price = float(input("How much: "))
