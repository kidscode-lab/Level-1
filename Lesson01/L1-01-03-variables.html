üîÑ Converting Input to Numbers

input() always gives you text - but what if you need numbers for math?

‚ö†Ô∏è

The Problem

input() always returns a string (text), even if you type numbers!

# This won't work for math!
age = input("How old are you: ")
# age is now "15" (text), not 15 (number)
‚ú®

The Solution

For whole numbers: int()
age = int(input("How old are you: "))
For decimal numbers: float()
price = float(input("How much: "))
